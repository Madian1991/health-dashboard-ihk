          # Health Dashboard – IHK Abschlussprojekt

Dieses Projekt ist mein Abschlussprojekt im Rahmen der IHK-Ausbildung
(Fachinformatiker – Anwendungsentwicklung).

## Projektbeschreibung
Das Health Dashboard ist eine Webanwendung zur übersichtlichen Darstellung
von Projekten, Aufgaben und Statusinformationen.
Ziel ist eine klare Trennung zwischen Backend und Frontend sowie eine
strukturierte und wartbare Architektur.

## Technologien
- Java
- Spring Boot
- REST-API
- HTML / CSS / JavaScript
- Maven

## Projektstruktur
- `source/` – Quellcode der Anwendung
- `docs/` – Projektdokumentation und Präsentation

## Dokumentation
Die vollständige Projektdokumentation und die Präsentation befinden sich
im Ordner `docs`.
